# Installation

## Stable release

To install Proteomics Pipeline Toolkit, run this command in your terminal:

```sh
uv add propit
```

Or if you prefer to use `pip`:

```sh
pip install propit
```

## From source

The source files for Proteomics Pipeline Toolkit can be downloaded from the [Github repo](https://github.com/fabianegli/propit).

You can either clone the public repository:

```sh
git clone git://github.com/fabianegli/propit
```

Or download the [tarball](https://github.com/fabianegli/propit/tarball/master):

```sh
curl -OJL https://github.com/fabianegli/propit/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd propit
uv pip install .
```
